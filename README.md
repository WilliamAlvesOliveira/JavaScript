# ⚡ JavaScript — Estudos, Prática e Evolução

Este repositório reúne meus estudos em **JavaScript**, com foco na construção de lógica, manipulação da linguagem e desenvolvimento de aplicações interativas.

Mais do que seguir apenas um curso, este espaço representa minha evolução prática, organização de aprendizado e experimentação com a linguagem.

---

## 🚀 Sobre

JavaScript é a linguagem responsável por trazer **interatividade para a web**, permitindo manipulação de elementos, eventos e dados em tempo real.

Neste repositório, utilizo a linguagem não apenas para aprender sintaxe, mas para desenvolver raciocínio lógico e compreensão prática do funcionamento da programação.

---

## 🧠 Abordagem de aprendizado

Durante os estudos, meu foco é:

* Entender como a linguagem funciona de fato
* Reescrever exemplos com minha própria lógica
* Testar diferentes soluções
* Explorar ideias além do conteúdo proposto
